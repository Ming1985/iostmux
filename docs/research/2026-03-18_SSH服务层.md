## Research 发现
- Citadel 完全支持交互式 shell（withPTY），之前 brainstorming 阶段认为"不支持"是错误的
- API 全部 async/await，与 SwiftUI 配合良好
- 核心方法：SSHClient.connect() → executeCommand() / withPTY()
- withPTY 使用 closure context，提供 TTYOutput（AsyncSequence）+ TTYStdinWriter
- 支持 terminal size change: ttyStdinWriter.changeSize()
- 注意: withPTY 标记为 @available(macOS 15.0, *)，需确认 iOS 对应版本

## 关联文件
- 新增: iostmux/Models/Project.swift, iostmux/Services/SSHService.swift

## 风险与边界
- withPTY 的 availability 标记可能需要 iOS 18+
- withPTY 是 closure-based，shell 生命周期与 closure 绑定，需要额外设计来保持 shell 跨视图存活

## 技术方案
- 目标: SSHService 服务层，支持命令执行和交互 shell
- 方案: Citadel async/await API。executeCommand 做单命令，withPTY 做交互 shell。密码认证占位。
- 边界: 不做 UI，不做 key auth，不做 reconnection
- 交付层级: 纯服务层代码，能编译通过
- 验收标准: xcodebuild build 成功
